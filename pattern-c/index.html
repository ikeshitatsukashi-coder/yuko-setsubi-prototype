Message

和気藹々と、
いい仕事を、やる。

友好設備は、約20名（事務含む）の小さな会社です。1980年、漏水修理業として2人で創業。一度ビル工事で大きな赤字を経験し、強みである「水道」に立ち戻ってからの46年です。いま、川口の上下水道は私たちが整えた配管が支えています。

大切にしているのは、「和気藹々と、いい仕事を、やる」。仕事は厳しく、でも雰囲気は和やかに。勤続10年以上が半数、最長40年という関係性が、それを証明しています。

採用で見るのは、資格の有無より「冗談が通じるか」「コミュニケーションが取りやすいか」。未経験でも、本気でやる気がある人ならぜひ会いに来てください。

工事部長
竹田　茂
