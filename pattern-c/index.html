A Day

1日の流れ

  1. 7:30

    出社・朝礼

    本日の現場と段取り確認。安全唱和。社長も毎日参加します。

  2. 8:00

    現場入り

    各チームで現場へ移動。社用車利用、移動手当あり。

  3. 12:00

    昼休憩

    近所のお弁当・定食屋で休憩。手当別途あり。

  4. 13:00

    午後の作業

    施工・進捗共有。安全第一、無理はさせません。

  5. 17:30

    帰社・片付け

    道具の手入れ、翌日の準備、報告書作成。残業はほぼなし。

  6. 18:00

    退社

    定時で上がれる現場が大半。プライベートを大切に。

Message

「手に職」って、
カッコイイ。

友好設備は、20人の小さな会社です。でも、川口の街の地面の下では、私たちが整えた配管や舗装が、毎日数十万人の暮らしを支えています。

若い人にこそ、この仕事の意味を知ってほしい。「手に職」は、これからの時代いちばん強い武器です。

未経験でも、年齢を重ねていても、本気でやる気がある人ならぜひ会いに来てください。社長の私が、直接お話しします。
